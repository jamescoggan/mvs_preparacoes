# Design Direction

## Concept

Redline Laboratory: a precise, technical, performance-focused site that turns the existing piston logo and red/black identity into a polished workshop experience.

## Audience Rationale

Visitors are likely considering a high-cost, high-trust automotive project. They need confidence that MVS understands custom performance work, a clear path for their project type, and practical contact details.

## Emotional Journey

1. Arrival: serious performance specialist, not a generic repair shop.
2. Orientation: clear services for street, track, motor, injection, and peripherals.
3. Confidence: visible process from briefing to execution and delivery.
4. Action: a concise brief and direct contact options.

## Visual System

- Typography: system sans-serif for speed and crisp technical readability.
- Color: near-black workshop base, MVS red, steel gray, white, and restrained amber/cyan accents for instrumentation.
- Composition: full-width bands, offset technical panels, dense but readable grids, no nested cards.
- Shape language: sharp panels, thin rules, compact chips, 8px or smaller radii.
- Image treatment: use the verified MVS logo as the primary brand asset; avoid fake workshop/project photography.
- Motion: subtle entrance reveals, scroll progress line, mobile menu state transitions. All disabled or simplified with `prefers-reduced-motion`.

## Signature Moments

- Hero logo and redline instrumentation panel.
- Service grid structured as project lanes rather than generic feature cards.
- Local-only project brief that builds email/WhatsApp messages without pretending to submit to a backend.

## Anti-Patterns

- No invented horsepower numbers, testimonials, awards, or project photos.
- No stock-looking workshop imagery.
- No blue generic CTA.
- No hidden mobile navigation.
- No decorative motion that delays essential content.
